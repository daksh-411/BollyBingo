# BollyBingo - A BollyWood Quiz Game
BollyBingo is a CLI based BollyWood movie guessing game. A movie title is randomly generated and user gets 5 attempts to guess the movie. Each wrong guess eliminates one attempt. The game continues until the player either guesses the movie correctly or all attempts are exhausted.

## Instructions
- A movie title is randomly generated from the csv file.
- Each letter in the movie is hidden with an underscore(_).
- Guess one letter at a time.
- Correct guesses reveal the letter in the title.
- Wrong guesses reduce your remaining attempts by one.
- You get 5 attempts, guess the movie before attempts become 0 to win!!!!

## Requirements
- Python 3.x
- Pandas Library

Install Pandas if required
```bash
pip install pandas
```

## How to Run the Game
1. Clone the Project
2. Open terminal in the project directory.
3. Run the script
   ```bash
   python main.py
   ```
